# Research Docs

These documents are research inputs, synthesis artifacts, and planning support
material.

They are useful context, but they are **not** canonical product truth.

Canonical authority remains:

- `docs/FEATURES.md`
- `docs/VISION.md`
- `docs/STATUS.md`
- `docs/ARCHITECTURE.md`
- `docs/RUNTIME_API.md`
- `docs/GROUND_TRUTH.md`

If a research document conflicts with code, tests, or canonical docs, the
research document loses.

Use this directory for:

- research prompts
- external-system comparison notes
- research synthesis documents
- architecture exploration that has not yet been adopted into product docs
